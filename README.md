### Project pre-requisites
`Rails 6.1.7`
`Gem 3.2.33`
`Ruby 3.0.4p208`
`Yarn`
`Npm`
`MySql2`
`Ubuntu 20.04.5 LTS`

### Cloning the repository
```sh 
$ git clone https://github.com/gbrlhnz/todolist_ruby
$ git pull
```
### Initializing the project for development
#### Backend
```sh 
$ cd backend/
$ bundler install
$ yarn
$ rails migrate:db
```
#### Frontend
```sh 
$ cd frontend/
$ npm i
```
